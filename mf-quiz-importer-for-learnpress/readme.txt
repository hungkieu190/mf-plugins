Import quizzes and questions into LearnPress LMS from CSV, Excel, and JSON files with ease.

== Description ==

**Quiz Importer for LearnPress** is a powerful and user-friendly plugin that makes it easy to import quizzes and questions into LearnPress LMS from various file formats.

= Key Features =

* **Dual Import Modes** - Import complete quizzes or add questions to existing quizzes
* **Multiple File Formats** - Support for JSON, CSV, and Excel (XLSX)
* **4 Question Types** - True/False, Single Choice, Multiple Choice, Fill in the Blanks
* **Modern Interface** - Beautiful UI with drag & drop file upload
* **Comprehensive Documentation** - 12 guides included with in-app viewer
* **Validation & Error Handling** - Detailed error messages and validation
* **Sample Files** - 6 sample files to help you get started

= Perfect For =

* Course creators importing quiz content
* Educators migrating from other LMS platforms
* Administrators managing bulk quizzes
* Developers creating quizzes programmatically

= File Format Support =

**JSON** - Best for complex data with nested questions
**CSV** - Easy to edit in Excel or Google Sheets
**Excel (XLSX)** - Native Excel format (Excel 2007+)

= Question Types =

* True or False
* Single Choice (one correct answer)
* Multiple Choice (multiple correct answers)
* Fill in the Blanks

Plus 20+ question type variations automatically mapped!

= Documentation =

The plugin includes comprehensive documentation:

All accessible via the in-app documentation viewer.

= Requirements =

* WordPress 5.8 or higher
* PHP 7.4 or higher
* LearnPress 4.0 or higher

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/mf-quiz-importer-for-learnpress` directory, or install through WordPress plugins screen
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Navigate to LearnPress → Quiz Importer
4. Download a sample file to see the format
5. Upload your quiz file and import!

== Frequently Asked Questions ==

= What file formats are supported? =

JSON, CSV, and Excel (XLSX) formats are fully supported. You can also open CSV files in Excel or Google Sheets for easier editing.

= Can I import questions to an existing quiz? =

Yes! The plugin has two import modes: Import Quizzes (creates new quizzes) and Import Questions (adds to existing quiz).

= How do I create the import file? =

Download a sample file from the plugin, open it in Excel or Google Sheets, modify with your content, and save as CSV or XLSX.

= What question types are supported? =

True/False, Single Choice, Multiple Choice, and Fill in the Blanks. The plugin also recognizes 20+ variations of these types.

= Is there a file size limit? =

Yes, the maximum file size is 10MB. This should be sufficient for most use cases.

= Can I import quizzes with questions in one file? =

Yes! You can use the advanced CSV/Excel format that includes both quiz metadata and questions in a single file.

= Where can I find sample files? =

Sample files are included in the plugin and can be downloaded from the Import Quizzes tab.

= Is Excel 97-2003 (XLS) supported? =

Limited support. We recommend using XLSX (Excel 2007+) or CSV format for best compatibility.
