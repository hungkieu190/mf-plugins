# Sound Files for LP Lesson Completion Sound

This directory should contain the following sound files:

## Required Sound Files

1. **ting.mp3** - Âm thanh kim loại nhẹ nhàng, sạch sẽ (Default)
   - Duration: < 0.5 seconds
   - Type: Light metallic sound
   - Suitable for: All audiences

2. **success-chime.mp3** - Âm thanh chiến thắng, vang vọng ngắn
   - Duration: < 0.5 seconds
   - Type: Victory chime with short echo
   - Suitable for: Creating sense of achievement

3. **magic-sparkle.mp3** - Âm thanh lấp lánh, nhẹ nhàng và huyền ảo
   - Duration: < 0.5 seconds
   - Type: Magical sparkle sound
   - Suitable for: Fun, relaxing atmosphere

4. **pop.mp3** - Âm thanh nổ nhỏ, năng lượng
   - Duration: < 0.5 seconds
   - Type: Small pop sound
   - Suitable for: Youthful, energetic vibe

## Sound Requirements

- **Format**: MP3 (recommended) or WAV
- **Duration**: Less than 0.5 seconds
- **Quality**: Clear and pleasant to hear
- **Volume**: Normalized to avoid being too loud or too quiet

## Where to Get Sounds

You can find free sound effects from:
- https://freesound.org/
- https://mixkit.co/free-sound-effects/
- https://www.zapsplat.com/
- https://soundbible.com/

## Testing

After adding sound files, test them by:
1. Activating the plugin
2. Going to LearnPress Profile → Âm thanh & Hiệu ứng
3. Selecting each sound option
4. Completing a lesson to hear the sound

## Note

The plugin will work without sound files, but no audio will play. The confetti effect will still work if enabled.
